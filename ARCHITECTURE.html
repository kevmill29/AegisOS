The one trick

The handoff — how the machine changes hands

Every other OS keeps doing its own thing while you game: indexing, updating, phoning home. Aegis is built around a single reflex — notice the game, then disappear. Three moving parts, one clean pass:

Watch

A tiny agent keeps watch

A Rust daemon runs in the background on under 1% of one core. It watches for the exact moment Steam starts a game through Proton — the way you'd notice a door open.

Hand off

Sleep Mode kicks in

The instant a game launches, the assistant dims to sleep and every non-essential OS task is squeezed into a corner of the CPU. The game gets the CPU, memory, and GPU — almost all of it.

Restore

Quit, and it wakes

Close the game and everything comes back to life instantly — the sphere brightens, the assistant is listening again, right where you left it. Nothing to manage.
